## 🚀 Как запустить

### Способ 1: Через терминал

```bash
# Клонировать репозиторий
git clone https://github.com/whunotexplain/java_oop_college.git

# Перейти в нужную папку
cd java_oop_college/laba_1

# Скомпилировать файл
javac discriminant.java

# Запустить программу
java discriminant
