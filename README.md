## 📋 Описание

Репозиторий с лабораторными работами по курсу **Java OOP**. Здесь собраны основные задания и дополнительные задачи по программированию на Java.
В src будут директории с названиями лабораторных,в них будут разделения на дополнительные и обязательные задания, а также будет /laba_name/.md, где будут детальные объясненения по поиску заданий в репе
